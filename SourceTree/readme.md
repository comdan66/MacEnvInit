# Source Tree
	* 下載 - [https://www.sourcetreeapp.com/](https://www.sourcetreeapp.com/)
	* 把它移到應用程式資料夾，並開啟
	* 同意，下一步
	* 登入，使用 Google 登入，下一步
	* 選擇 `Skip Setup`

## 設定
	* 快捷鍵 `command + ,`
	* `General` > `Project folder` > 選擇你放專案的資料夾
	* `General` > `Terminal app` > 選擇 `iTerm2`
	* `Commit` > 下方可以填寫懶的填註解的預設文字
